# 快速上手指南

## 第一步：启动项目

```bash
cd "C:/Users/86199/Desktop/ai互动"
npm run dev
```

浏览器会自动打开 `http://localhost:3006`

## 第二步：配置 API Key

1. 点击右上角的 ⚙️ 设置按钮
2. 输入 GPT 或 Claude 的 API Key
3. 点击"保存"

### 获取 API Key

**GPT-4o (OpenAI)**:
- 访问 https://platform.openai.com/api-keys
- 注册账号并创建 API Key

**Claude 3.5 (Anthropic)**:
- 访问 https://console.anthropic.com/
- 注册账号并创建 API Key

## 第三步：开始对话

1. 点击右下角的 💬 按钮打开对话窗口
2. 在输入框输入文字
3. 按回车或点击"发送"
4. 高木同学会回复你！

## 功能说明

### 语音功能

- 点击标题栏的 🔊 按钮开启/关闭语音
- 开启后，高木同学的回复会自动朗读
- 朗读时模型口型会自动同步

### AI 模型切换

- 在对话窗口标题栏可以切换 GPT/Claude
- 两个模型都支持流式输出

### Live2D 互动

- **视线追踪**: 移动鼠标，模型会跟随你的视线
- **点击互动**: 点击模型会触发反应
- **口型同步**: 语音播放时口型会自动变化

### 环境系统

- 背景会根据时间自动切换
- 高木同学会根据时间打招呼

## 常见问题

### Q: 页面空白怎么办？

A: 检查浏览器控制台是否有错误，确保网络连接正常。

### Q: 模型不显示怎么办？

A: 确保 `live2dcubismcore.min.js` 文件存在，尝试刷新页面。

### Q: 语音不工作怎么办？

A: 检查浏览器是否支持 Web Speech API，尝试使用 Chrome 浏览器。

### Q: API 调用失败怎么办？

A: 检查 API Key 是否正确，确保账户有足够余额。
